About MentorConnect

MentorConnect is a platform I created for connecting mentors and mentees, while following a Udemy course.It lets users set up their profiles, browse other members, follow/unfollow each other, and message each other in real-time. Feel free to try it out!

Built with:
  • Frontend: Angular 16 (TypeScript-based framework)
  • Backend, ORM & Authentication: ASP.NET Core 7, Entity Framework, ASP.NET Core Identity
  • Database, Containerization & Hosting: PostgreSQL, Docker, Render.com
  • CSS Styling: Bootstrap 5 (CSS framework, Angular support provided by ngx-bootstrap)
  • Services: SignalR (real-time presence tracking & messaging), Cloudinary (photo hosting)
  • Other: HTML, CSS, RxJS, Bootswatch, FontAwesome Icons, ngx-gallery, ng2-file-upload
