# MentorConnect
A web app to connect mentors and mentees in order to combat imposter syndrome. The web app was only partially completed due to time constraints.

I made this web app for DeltaHacks VI, a hackathon at McMaster University from Jan. 25-26, 2020 (https://www.deltahacks.com/). You can see it live at https://mentorconnect-621bb.firebaseapp.com.
